# 一、澜起科技（688008 / 06809.HK）

## 1.1 一句话 Thesis

澜起科技是全球 **内存接口芯片龙头**（市占 ~37%），正从「DDR5 接口」向「AI 服务器高速互连平台」升级。**行业顺风、公司优质，但 A 股 ~264 元已把 2–3 年成长大幅定价——长期值得关注，当前价位不适合新进场。**

---

## 1.2 公司背景

| 项目 | 内容 |
|------|------|
| 全称 | 澜起科技股份有限公司 |
| 代码 | A 股 **688008** / H 股 **06809**（2026/2 上市） |
| 成立 / 上市 | 2004 年 / 2019 年科创板 |
| 模式 | Fabless（台积电代工） |
| 地位 | 全球内存接口芯片第一，JEDEC 董事会唯一中国芯片企业 |
| 创始人 | 杨崇和（海归半导体资深专家） |

### 装机 / 产品结构

```
互连类芯片（94% 收入）
├── DDR5 RCD/DB          ← 基本盘，子代持续迭代
├── MRCD/MDB（MRDIMM）   ← AI 服务器最大增量，单模组芯片 ×11
├── PCIe Retimer         ← 第二曲线，AI 服务器 8–24 颗/机
└── CXL MXC              ← 第三曲线，内存扩展，尚在验证期
```

---

## 1.3 盈利模式

```
利润 ≈ 芯片出货量 × 单颗价值量 × 毛利率 − 研发费用

单颗价值量提升路径：
  标准 RDIMM（1 颗 RCD）→ MRDIMM（1 MRCD + 10 MDB = 11 颗）
  + DDR5 子代升级（4800 → 8000+ MT/s）
  + PCIe 5.0/6.0 Retimer 放量
```

| 维度 | 内容 |
|------|------|
| **客户** | 三星（~28%）、SK 海力士、美光 + 戴尔/HPE/华为 |
| **壁垒** | JEDEC 标准制定、1–2 年认证周期、全球寡头 CR3>93% |
| **竞争** | 瑞萨（~30%）、Rambus（~26%），市占差距仅 ~7pct |
| **风险** | 客户集中（前五大 ~77%）、存储周期、竞争加剧 |

## 2.4 盈利模式差异

### 大唐：北方煤电 + 风光增量 + 算电叙事

```
收入 = 北方火电 ×（电价 − 煤价）+ 风光水增量 + 核电参股
优势：煤产区布局，度电燃料成本低；2025 弹性最大
劣势：负债 70%；算电协同未兑现；A 股易被炒作
```

### 华能：全国规模 + 成本领先 + 海外

```
收入 = 全国煤电 + 风光（45GW+）+ 燃机 + 海外（新加坡、巴基斯坦）
优势：规模最大、煤耗 292g/kWh 行业领先、股息 A 5.3% / H 7.7%
劣势：2026 长协电价普降，体量大受冲击也大
```

### 浙能：浙江区域垄断 + 核电投资收益

```
利润 ≈ 浙江火电（85%）+ 核电/火电参股收益（~39 亿，占利润一半）
优势：区域垄断、分红率 59%、负债低
劣势：2026 浙江长协电价 -16.4%；几乎无新能源转型
```

# 三、跨案例对照（Section 1 框架应用）

| 框架环节 | 澜起科技 | 电力三家（共性） |
|----------|----------|------------------|
| **行业趋势** | AI + DDR5，强顺风 | 煤价红利消退，电价下行，中性偏逆风 |
| **公司质量** | 优秀（高毛利、低负债） | 华能/浙能优，大唐负债高 |
| **估值** | **极贵**（PE ~98x） | 华能/浙能合理，大唐爆炒后偏贵 |
| **行情** | 题材后段，股东减持 | 大唐题材退潮，华能/浙能平稳 |
| **长持** | 公司等价格 | 华能 > 浙能 > 大唐（看价位） |
| **短持** | 不追 | 大唐短期风险 > 机会 |

### 共同教训

1. **好公司 ≠ 好股票** — 澜起、大唐都是「逻辑对、价格错」的典型案例  
2. **长/短标准分开** — 大唐 7 元不宜长持，澜起 264 元不宜短追  
3. **先 thesis，再估值，再时机** — 三家电力中华能同时满足三个条件；澜起和大唐只满足第一个
