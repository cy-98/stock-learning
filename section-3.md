# Section 3：五层模型股票分析与合理价格区间

> 基于 Section 1 方法论框架，对五层模型中的候选股票进行系统化分析。数据参考截至 **2026 年 5 月下旬**，不构成投资建议。

---

## 分析方法说明

本文按 Section 1 的分析链对每只股票进行评估：

```
行业趋势（顺风/逆风）→ 公司质量（商业模式 + 财务）→ 估值（合理价格区间）→ 持有建议
```

估值方法根据公司类型选择：
- **成长型芯片设计**：PE/PEG，参考历史分位与同行
- **周期型半导体**：PB + 景气位置
- **电力公用事业**：股息率 + PB + PE
- **消费龙头**：PE，参考历史中枢
- **资源股**：商品价格弹性 + PB
- **美股科技**：PE/PEG + FCF

# 使用说明

## 如何使用本表

1. **对照当前价格**：将实时行情与"合理区间"对比，判断贵/合理/便宜
2. **关注关键变量**：每只股票的"关键变量"是证伪/确认 thesis 的核心
3. **分层配置**：根据风险偏好在不同层级分配仓位
4. **动态更新**：每季报后更新 EPS 预测，重新计算合理区间

## 重要提醒

- 合理价格区间 ≠ 必然达到的价格，市场可以长期偏离合理估值
- 遵守 Section 1 纪律：**先 thesis → 再估值 → 再时机**
- 长持标准与短持标准分开（参见 Section 2 案例教训）
- **证伪条件比买入理由更重要**——写好什么情况下认错
