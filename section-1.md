# Section 1：股票投资分析入门

> 整理自学习讨论。内容为方法论分享，不构成投资建议。

---

## 一、你的思路 Review

### 你想做的事

```
最新行情 → 公司运转逻辑 / 背景 → 时代趋势 → 判断长持 or 短持
```

### 结论：**方向是对的**

这是专业研究的基本路径，本质上是 **自上而下 + 自下而上** 的结合：

| 你的环节 | 对应的专业能力 | 评价 |
|----------|----------------|------|
| 了解最新行情 | 市场感知（价格、预期、资金） | ✅ 必要，但需区分「事实」与「叙事」 |
| 公司运转逻辑 / 背景 | 基本面分析（商业模式、财务） | ✅ 核心能力，长期判断的根基 |
| 时代趋势 | 宏观 + 行业研究 | ✅ 决定顺风还是逆风 |
| 长持 / 短持判断 | 估值 + 催化剂 + 风险管理 | ✅ 正确，但两种决策标准应分开 |

### 需要补强的三点

1. **估值环节不能省**  
   好公司 ≠ 好股票。趋势对、逻辑对，买贵了仍会亏。要在分析链中加入：**当前价格是否合理**。

2. **长持和短持是两套标准，不能混用**  
   - 长期：看 3–5 年后公司是否更强  
   - 短期：看未来 1–3 个月有没有催化剂、预期差  
   同一只股票可以「长期看好、短期观望」，反之亦然。

3. **行情是起点，不是结论**  
   不要从涨跌反推逻辑。正确顺序是：**先建立 thesis（投资论点）→ 再看市场是否已定价 → 最后决定时机**。

### 修正后的分析链

```
宏观 / 行业趋势（顺风？）
        ↓
公司商业逻辑 + 财务质量（能不能持续赚钱？）
        ↓
估值（当前价格贵不贵？）
        ↓
行情 / 催化剂（现在进还是等？）
        ↓
定持有周期 + 仓位 + 证伪条件（错了怎么办？）
```

## 二、核心框架：三层分析法

```
宏观环境 → 行业趋势 → 公司个体 → 估值与时机 → 持有决策
```

| 层级 | 核心问题 | 主要信息源 |
|------|----------|------------|
| 宏观 | 利率、政策、经济周期处在哪？ | 央行报告、GDP/CPI、产业政策 |
| 行业 | 扩张、洗牌还是衰退？ | 行业研报、渗透率、竞争格局 |
| 公司 | 怎么赚钱？护城河在哪？ | 年报、招股书、投资者交流纪要 |
| 估值 | 贵还是便宜？预期是否过高？ | PE/PB、同行对比、历史分位 |
| 时机 | 短期催化剂或风险？ | 业绩预告、政策节点、资金面 |

## 三、五项核心技能

### 1. 读行情——分清三层信息

| 层级 | 看什么 |
|------|--------|
| 价格 | 涨跌幅、成交量、相对大盘强弱 |
| 预期 | 市场在交易什么故事？新逻辑还是旧题材？ |
| 资金 | 谁在买谁在卖？（机构、北向、游资） |

**每日练习：** 选 1 只股票，写 3 句话——今天为什么动？行业带动还是个股消息？是新故事还是老故事？

---

### 2. 懂公司——一张画布讲清逻辑

```
客户是谁 → 卖什么 → 怎么收费 → 成本结构 → 竞争壁垒 → 增长靠什么
```

**必看材料（按优先级）：**

1. 年报「管理层讨论与分析」
2. 招股书（新公司历史最全）
3. 分业务 / 分地区收入与毛利率
4. 前五大客户 / 供应商（依赖风险）
5. 投资者关系活动记录

**长期质量四指标：**

| 指标 | 好公司的常见特征 |
|------|------------------|
| 毛利率 / 净利率 | 稳定或改善，同行偏上 |
| ROE | 长期 > 15%，非靠高杠杆堆出 |
| 经营现金流 | 长期接近或高于净利润 |
| 负债率 | 行业合理范围内，无隐性风险 |

---

### 3. 看趋势——判断顺风还是逆风

关注 **结构性变量**，而非追热点：

- **渗透率**：5% → 20% 常是行业爆发期
- **政策方向**：扶持 / 限制 / 国产化等
- **技术拐点**：成本下降、新场景落地
- **竞争格局**：龙头集中 vs 群雄混战

**PEST 速查：** Policy（政策）· Economy（经济）· Society（需求）· Technology（技术）

---

### 4. 估价格——贵不贵

| 方法 | 适用场景 |
|------|----------|
| PE（市盈率） | 盈利稳定的成熟公司 |
| PEG | 成长股（PE ÷ 增速） |
| PB（市净率） | 银行、周期、重资产 |
| DCF（现金流折现） | 长期深度研究（进阶） |

- **长期买入**：合理或偏低估值 + 优质公司  
- **短期参与**：预期差（市场尚未充分定价的好消息，或过度反应的错杀）

---

### 5. 定周期——长持 vs 短持标准分开

| 维度 | 长期（数月～数年） | 短期（几天～几周） |
|------|-------------------|-------------------|
| 核心问题 | 5 年后公司更强还是更弱？ | 未来 1–3 个月有什么催化剂？ |
| 失败原因 | 商业模式恶化、行业被替代 | 预期落空、情绪退潮、利好出尽 |
| 退出逻辑 | thesis 被证伪（逻辑变了） | 时间止损 + 价格止损 |
| 信息权重 | 财报、行业格局、管理层 | 业绩指引、订单、政策节点 |

## 四、投资备忘录模板

每次分析固定用这个结构，练 20 次后会明显进步：

```markdown
## 1. 一句话 thesis
我认为 ___ 在未来 ___ 因为 ___

## 2. 行业与趋势（顺风 / 逆风）
## 3. 商业模式与护城河
## 4. 财务健康度（3 个最关键指标）
## 5. 估值：贵 / 合理 / 便宜，依据是什么
## 6. 短期催化剂与风险（未来 3 个月）
## 7. 长期证伪条件（什么情况下我认错）
## 8. 计划：长持 / 短持 / 观望，仓位与止损
```

**证伪条件** 最重要：提前写好「什么情况下说明我错了」，避免死扛。

## 五、学习路径（3–6 个月）

| 阶段 | 时间 | 任务 |
|------|------|------|
| 工具与语言 | 第 1 月 | 学会读三张财报表；读《手把手教你读财报》 |
| 行业与公司 | 第 2 月 | 深度跟 1 个行业 + 3 家公司，各输出 1 页商业逻辑 |
| 宏观与估值 | 第 3 月 | 跟踪利率 / 政策 1 个月；给 3 只股票写买入 / 观望 memo |
| 刻意练习 | 第 4 月起 | 模拟盘或极小仓位；每周复盘：错在信息、逻辑还是情绪 |

**最高效训练：** 选 3–5 只不同行业股票，连续 90 天跟踪——读公告、写 memo、周末复盘。

## 六、入门纪律

### 信息可靠度（从高到低）

1. 交易所公告、年报、审计报告  
2. 公司官方投资者关系材料  
3. 主流券商深度报告（交叉验证）  
4. 财经媒体与社群（过滤情绪与叙事）

### 操作原则

- 不借钱、不加杠杆  
- 不 all in 单只股票  
- 不追热点、不听「内幕消息」  
- 重大结论至少 2 个独立来源印证  
- 进场前定好持有周期与退出条件  

### 常见误区

| 误区 | 正确做法 |
|------|----------|
| 从涨跌反推逻辑 | 先逻辑，再看价格是否反映 |
| 把故事当业绩 | 用现金流和订单验证 |
| 长期短做、短期长扛 | 长 / 短标准分开，不混用 |
| 信息越多越好 | 围绕 thesis 建清单，避免焦虑刷屏 |

